Удалённый файловый менеджер

File centipede представил нативный пользовательский интерфейс для удалённой загрузки файлов.

Он находится в меню Удалённый.

Он очень прост в использовании. Вам нужно просто указать IP либо домен удалённого хоста, и нажать на хост из только что вами добавленного элемента меню.

Если вы хотите, чтобы к вашему ПК могли подключаться, откройте Настройки -> Сервис и включите параметр Удалённый доступ, и не забудьте добавить пароль.

Команды для командной строки

filec remote_access on

filec service_port 10111

filec service_password your_password
